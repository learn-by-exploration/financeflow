# Iteration 3 — v0.1.3: Budgets & Goals Views

**Date:** 2026-03-30 | **Sprint:** 3

## Expert Panel Summary

| Role | Key Insight |
|------|-------------|
| **UI/UX** | Progress bars with color transitions (green→yellow→red at 80%/100%); goals show projected completion date |
| **Architect** | Budget summary endpoint already provides allocated vs spent per category — render directly |
| **QA** | Test edge cases: zero-amount budgets, 100%+ spent display, goals with no deadline |
| **PM** | Users see budget progress at a glance, can drill into category spending; goals show clear path to target |

## Scope

1. **Budgets View** — List budgets, create with category allocations, view summary with progress bars
2. **Goals View** — Card layout with progress rings, add/edit/delete, contribute to goals

## Acceptance Criteria

- [ ] Budget list shows all budgets with period and date range
- [ ] Create budget with name, period, date range, category allocations
- [ ] Budget summary: per-category allocated vs spent with progress bars
- [ ] Goals cards with progress percentage and visual bar
- [ ] Add/edit/delete goals with target amount, current amount, deadline
- [ ] Goals auto-complete when current >= target
- [ ] Empty states for both views
